Change 7 — Fix upgrade button visibility for paid plan users in ProjectHub

**File: `src/ui/platform/ProjectHub.ts`**

Current line 80:

```
${(this.user.plan||'free')==='free'?`<button class="ph-upgrade-btn" id="ph-upgrade-btn">✦ Upgrade</button>`:''}
```

This hides the upgrade button for Architect and Studio users, giving them no visible upgrade path to a higher tier.

Replace with:

```
${(this.user.plan||'free')!=='enterprise'?`<button class="ph-upgrade-btn" id="ph-upgrade-btn">${(this.user.plan||'free')==='free'?'✦ Upgrade':'✦ Manage plan'}</button>`:''}
```

This shows the button for all plans except Enterprise. Free users see "✦ Upgrade". Paid users (Architect, Studio, Firm) see "✦ Manage plan". The click handler at line 235 already calls `this.callbacks.onUpgrade?.()` — no change needed there.
